#Crypto Currency with Flutter

## Features
- API(Nomics)
- Search
- Graph(with fake data)
- Coin Details (1D,1W,1M,1Y,ALL)

## Stack
- Flutter 2.10.3
- Dio
- Bloc(Cubit)
- Get It
- Syncfusion Futter Charts
